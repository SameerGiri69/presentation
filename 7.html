UDP Multicast

Instead of configuring IP addresses manually, the application "shouts" its presence to a specific multicast group address. Any other instance listening on that address instantly detects the peer.

  • Zero Configuration

    Works instantly on the local network. No server setup or IP typing required.

  • Periodic Announcements

    Broadcasts every 1-3 seconds. If a peer stops broadcasting, it's removed from the "Nearby" list.

  • Minimal Overhead

    Small JSON packets (under 1KB) sent via UDP keep network traffic negligible.
