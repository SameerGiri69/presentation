Performance & Design

Key Technical Highlights

LANGUAGE

Go 1.21+

Minimal Dependencies

Built with a "standard library first" philosophy. Only essential external packages are used to keep the binary small and secure.

Binary Size < 15 MB

Stable Memory Usage

Fixed 64KB chunk buffer ensures RAM usage remains constant regardless of file size (1MB vs 10GB).

RAM Usage vs File Size

Vanilla JS Frontend

Zero frameworks (No React/Vue). Pure HTML/CSS/JS for instant load times and minimal browser overhead.

Bundle Size ~12 KB

Reliable Delivery TCP/IP

Leverages TCP's built-in error checking, ordering, and retransmission mechanisms. Ensures files arrive bit-perfect without complex application-level checksums for every packet.

Ordered Packets Flow Control

Real-time UX WebSockets

Event-driven architecture pushes updates immediately. Discovery of new peers and transfer progress happens instantly without the user needing to refresh.

Latency < 10ms
