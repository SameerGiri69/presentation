Roadmap

Future Enhancements

Planned for v2.0
Security & Network
Priority: High

E2E Encryption (TLS)

Upgrade TCP streams to TLS 1.3 to prevent man-in-the-middle snooping on public Wi-Fi. Ensure file contents remain private during transit.

crypto/tls Self-signed Certs
Reliability
Priority: High

Pause & Resume

Implement transfer checkpoints. If a connection drops or user pauses, resume from the last successful 64KB chunk offset instead of restarting.

Seek() Range Requests
User Experience
Priority: Med

Multi-file & Folders

Support queuing multiple files and recursive folder transfer. Compress small files into a single zip stream or handle sequential TCP sessions.

archive/zip Queue Logic
Priority: Med

Cross-Subnet Support

Move beyond local broadcast. Implement STUN/TURN integration or a lightweight relay server to allow transfers between different subnets or over the internet.

pion/webrtc Relay
Priority: Low

Bandwidth Throttling

Allow users to limit transfer speed (e.g., "Max 5 MB/s") to prevent saturating the network while gaming or streaming video on the same LAN.

rate/limiter Token Bucket
Priority: Low

Admin Metrics Dashboard

A dedicated view for visualization of transfer history, total bandwidth usage, error logs, and active peer connection stats over time.
