Context & Strategy

Project Overview & Motivation

The Problem

Moving large files on LAN is clumsy. USB drives require physical swapping, chat apps compress media, and cloud solutions waste bandwidth on round-trips.

The Goal

Create a fast, secure, local-only transfer system with real-time visibility. No internet required, no file size limits, and instant peer discovery.

The Approach

UDP Multicast for "Radar" discovery, TCP streaming for reliable chunked data transfer, and WebSockets for live progress updates.

The Philosophy

"Minimal Dependencies." Built primarily with standard Go packages (net, io, context) to ensure a lightweight footprint and easy deployment.
