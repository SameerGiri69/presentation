Dependencies

Libraries Used

Selected for stability, security, and performance.

github.com/google/uuid

Generates collision-free unique IDs for every file transfer session, ensuring reliable tracking.

github.com/gorilla/websocket

Enables real-time, full-duplex communication between the Go server and Web UI for progress updates.

github.com/lib/pq

Pure Go driver for PostgreSQL. Handles metadata storage, user records, and transfer logs.

golang.org/x/crypto/bcrypt

Securely hashes user passwords using salt and adaptive cost factors before database storage.

gopkg.in/gomail.v2

Simple, efficient SMTP client for sending email notifications upon transfer completion.

"We use standard library packages (net, io, context) for 90% of the functionality to keep the binary small and secure."
