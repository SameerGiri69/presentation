import "golang.org/x/crypto/bcrypt" func HashPassword(password string) (string, error) { bytes, err := bcrypt.GenerateFromPassword( []byte(password), bcrypt.DefaultCost, // 10 ) return string(bytes), err }

  • Passwords are never stored in plaintext.

  • Slow hashing algorithm protects against brute-force attacks.
